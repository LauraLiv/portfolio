WEB

Grundlæggende Web var vores første forløb. Vi fik en faglig forståelse af multimediedesignerens rolle. Vi opsatte vores eget domæne og herfra begyndte vi at kode. Vi arbejdede med kodning på VSCode, hvor vi brugte HTML og CSS til at opbygget og style hjemmesider. Forløbet blev inddelt i to opgaver. Første opgave var Website også kaldet Studiestartsprøven. Dette var en hjemmeside der skulle kodes på baggrund af et given wireframe, layoutdiagram, tekst samt billeder. Anden del af forløbet var Emnesite. Denne opgave var således en hjemmeside der skulle kodes ved hjælp af HTML og CSS. Denne gang skulle vi selv vælge et emne, og kunne style websitet som vi ønskede. Dette gjorde også at vi skulle vise de færdigheder vi havde lært fra tidligere opgave.

02.01.02 Website (Studiestartsprøven)

Denne opgave var vores første kodningsprojekt. Jeg brugte henholsvis HTML og CSS til opbygning og layout. Vi fik mulighed for selv at bestemme styling. Jeg brugte CSS til denne del, hvor jeg valgte nogle neutrale jordfarver og bløde kanter som jeg kodede ved at dedfinere color og border radius. Som font family brugte jeg Kanit og Varela. Først nævnte er mere kraftig og derfor brugte jeg den til overskifterne (h1/h2/h3), mens sidst nævnte er mere let og hermed valgte jeg at bruge den tilbrødtekst (p).

02.02.01 Emnesite

Anden opgave var således en hjemmeside der skulle kodes ved hjælp af HTML og CSS. Den skulle laves på baggrund af et emne vi selv havde valgt. Jeg havde på dette tidspunkt netop boet et år i Berlin og tænkte at jeg ville skrive om denne oplevelse. Det vigtige for mig var det visuelle udtryk, store billeder og stor tekst, som er kendetegn på stilarten Digital Modernisme. Jeg brugte billeder taget på mit digitale kamera og satte dem i fokus, og til selve hjemmesiden brugte jeg sort/hvid, som er to kontrastfarver, så billederne var mere i øjenfaldene.
